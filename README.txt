Commandes ------------
Joueur bleu :
- z,q,s,d pour bouger
- espace pour tirer (possibilité de charger un tir)
- b pour activer le bouclier (actif pendant 5 secs, 20 secs de cooldown)

Joueur rouge :
- flèches directionnelles pour bouger
- entrée pour tirer
- p pour activer le bouclier

Replay et IA désactivés pour éviter de générer n'importe quoi
